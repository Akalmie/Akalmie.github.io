My biggest Project :
The connected trash bin

End of study project for my bachelor's degree
The idea of setting up connected bins is to improve the trajectory of the collection trucks. To do this, we have to install various sensors (weighing, ultrasound, magnetic, location, etc.) on the bins, thus enabling the treatment centre to know their status. Indeed, there is no point in emptying a bin that is not considered full. The status of the bin is sent to the processing centre by means of a remote communication, which will then allow the trucks to know which route to take.

Language used : HTML, CSS, PHP, MySLQ, C++
